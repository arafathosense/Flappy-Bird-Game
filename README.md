# 🐦 Flappy Bird Game

A simple and fun Flappy Bird clone built using Python and OpenCV. Control the bird, avoid obstacles, and try to achieve the highest score!

## 🎮 Features

- Real-time gameplay
- Smooth bird movement and gravity physics
- Randomly generated pipes
- Collision detection
- Score tracking
- Simple and clean UI



## 🛠️ Technologies Used

- Python
- Pygame

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/arafathosense/flappy-bird-game.git
cd flappy-bird-game
````

2. Install dependencies:

```bash
pip install pygame
```


## ▶️ How to Play

Run the game:

```bash
python main.py
```

### Controls:

* Press **Spacebar** (or use your configured input) to make the bird jump
* Avoid hitting pipes or the ground
* Try to survive as long as possible!


## 🎯 Game Rules

* Each pipe passed = +1 score
* Collision with pipe or ground = Game Over
* Game restarts after failure
